## 为什么 Redmi AX6000 是目前的性价比天花板？

在折腾过各种复杂的 x86 软路由之后，很多人最终都会回归到 ARM 架构的硬路由方案。Redmi AX6000 凭借联发科 MT7986A 这颗神U，不仅发热量极低，而且在刷入第三方固件后，其 AES 加密解密性能足以跑满千兆带宽。

### ImmortalWrt 刷机核心思路

相比于原版 OpenWrt，**ImmortalWrt** 对国内网络环境和常用的科学插件有着更好的底层优化。

1. **解锁 SSH**：这是第一步，也是最繁琐的一步。你需要通过特定的 Python 脚本或工具，利用官方固件的漏洞开启 SSH 权限。
2. **刷入 uboot**：千万不要跳过这一步！刷入不死 uboot 是为了防止未来刷机变砖。即使固件崩溃，你也可以通过按住 Reset 键通电进入 uboot 的 Web 界面进行急救。
3. **刷入固件**：在 uboot 界面直接上传 ImmortalWrt 的 `.bin` 固件，等待 3 分钟重启即可。

### 差旅组网方案：把家里的网络装进口袋

当我们在酒店或咖啡厅时，连接公共 Wi-Fi 不仅不安全，而且无法访问家里的 NAS 或智能家居。

**完美的解决方案：携带一台 CUDY TR3000 迷你旅行路由器。**
这台机器极其小巧，专门为携带而生。它的核心玩法是：
1. **连接上游网络**：在酒店里，让 CUDY TR3000 通过网线或无线中继的方式连上酒店的 Wi-Fi。
2. **打通安全隧道**：在 CUDY 中内置 WireGuard 或 Tailscale 客户端，一键连回你家中的 Redmi AX6000 主路由。
3. **无感漫游**：你的手机、电脑连接 CUDY 发出的 Wi-Fi 信号，所有的网络流量都会加密传回你家中的路由器再发往互联网。你的 IP 地址永远是你家的宽带 IP，彻底告别异地登录的风控警告。
